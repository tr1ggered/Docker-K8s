# Топология учебного Kubernetes-кластера

## Назначение

Кластер состоит из одного control-plane узла и двух worker-узлов.

## Виртуализация и сеть

- Гипервизор: VMware Workstation
- Тип сети: VMnet8 NAT
- Подсеть: `192.168.110.0/24`
- Шлюз: `192.168.110.2`
- DHCP-диапазон: `192.168.110.128-192.168.110.254`
- Статические адреса узлов находятся вне DHCP-диапазона.

## Узлы

| Узел | Роль | IP-адрес | vCPU | RAM | Диск | ОС |
|---|---|---|---:|---:|---:|---|
| kube1 | control plane | `192.168.110.11` | 2 | 4 GiB | 30 GiB | Ubuntu Server 24.04.4 LTS |
| kube2 | worker | `192.168.110.12` | 2 | 3 GiB | 30 GiB | Ubuntu Server 24.04.4 LTS |
| kube3 | worker | `192.168.110.13` | 2 | 3 GiB | 30 GiB | Ubuntu Server 24.04.4 LTS |

Корневая файловая система каждого узла расширена до размера примерно
28 GiB с помощью LVM.

узла.
